# Лабораторна робота №5: Захищена електронна пошта

## Опис

Ця папка містить проєкт із графічним інтерфейсом (GUI) на Python, який реалізує систему симетричного шифрування повідомлень та файлів для захищеного листування.

* `main.py` - Головний файл запуску програми (графічний інтерфейс на Tkinter).
* `crypto_mail.py` - "Двигун", що містить авторську реалізацію алгоритму шифрування (XOR з гамуванням) та генерацію персональних ключів.

## Технічне завдання

Програма реалізує інструмент для захисту конфіденційності листування. Вона виконує наступні завдання, визначені у методичних рекомендаціях:
* **Власна реалізація алгоритму:** замість готових крипто-бібліотек реалізовано потоковий шифр на основі операції **XOR** з циклічним ключем (гамування).
* **Персоналізація:** ключ шифрування (256 біт) генерується на основі хешу (SHA-256) від персональних даних користувача (ПІБ, дата народження, пароль), що забезпечує його унікальність.
* **Кодування:** для зручної передачі зашифрованих повідомлень через електронну пошту використовується формат **Base64**.
* **Бонус (GUI):** розроблено повноцінний графічний інтерфейс на бібліотеці `tkinter` з вкладками.
* **Бонус (Робота з файлами):** реалізовано можливість шифрування та дешифрування будь-яких файлів (вкладень).
* **Бонус (Обмін ключами):** додано функціонал експорту та імпорту ключів у файл для обміну зі співрозмовником.

**Технологія:** Python 3, Tkinter (GUI), Hashlib, Base64 (стандартні бібліотеки).

## Запуск програми

Програма використовує лише стандартні бібліотеки Python, тому встановлення сторонніх пакетів не потрібне.

1.  Переконайтесь, що у вас встановлено **Python 3.x**.
2.  Перейдіть до поточної папки (`lab05`) у терміналі.
3.  Запустіть головний скрипт командою:
    ```bash
    python main.py
    ```
4.  Використовуйте графічний інтерфейс:
    * **Вкладка 1:** Згенеруйте свій ключ на основі даних або завантажте його з файлу.
    * **Вкладка 2:** Шифруйте та дешифруйте текстові повідомлення (використовуйте кнопки копіювання/вставки).
    * **Вкладка 3:** Оберіть файл для шифрування, щоб створити захищений архів `.enc`.
