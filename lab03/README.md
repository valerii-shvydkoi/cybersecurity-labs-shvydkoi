# Лабораторна робота №3: Невидимі дані у власних файлах

## Опис

Ця папка містить модульний проєкт на Python, який реалізує алгоритм стеганографії LSB (Least Significant Bit) для приховування та витягування секретних повідомлень із зображень.

* `main.py` - Головний файл для запуску програми (CLI).
* `steganography.py` - "Двигун", що містить всю "чисту" логіку LSB (hide/extract), шифрування та обробки байтів.
* `requirements.txt` - Список зовнішніх залежностей (Pillow).

## Технічне завдання

Програма реалізує повний цикл LSB-стеганографії. Вона виконує наступні завдання, визначені у методичних рекомендаціях:
* **Самостійна реалізація LSB:** програма вручну маніпулює молодшими бітами (LSB) пікселів зображення (`pixel[channel] & ~1) | int(bit)`).
* **Заборона бібліотек:** не використовуються готові бібліотеки стеганографії (як `stegano`).
* **Бонус (Маркер кінця):** реалізовано маркер `EOF_MARKER` ("||[EOF]||") для коректного витягування даних та уникнення "цифрового шуму".
* **Бонус (Шифрування):** повідомлення перед приховуванням шифрується **шифром Віженера** (використовуючи ключ-пароль), що значно підвищує безпеку.
* **Бонус (Кілька каналів):** алгоритм використовує всі 3 кольорові канали (R, G, B) для запису, збільшуючи ємність контейнера.

**Технологія:** модульний Python 3, Pillow (PIL).

## Запуск програми

Програма має одну зовнішню залежність (`Pillow`) для маніпуляції зображеннями.

1.  Переконайтесь, що у вас встановлено **Python 3.x**.
2.  Перейдіть до поточної папки (`lab03`) у терміналі.
3.  Встановіть необхідні залежності (рекомендується у віртуальному середовищі):
    ```bash
    pip install -r requirements.txt
    ```
4.  Запустіть головний скрипт командою:
    ```bash
    python main.py
    ```
5.  Дотримуйтесь інструкцій у меню:
    * Оберіть **"1. Приховати (Encode)"**, щоб вказати шлях до зображення, секретне повідомлення, ключ та ім'я нового файлу.
    * Оберіть **"2. Витягти (Decode)"**, щоб вказати шлях до стегоконтейнера та ключ для розшифрування.
