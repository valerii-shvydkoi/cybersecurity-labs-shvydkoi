# Лабораторна робота №7: Комплексний захист особистого проекту

## Опис

Ця папка містить фінальний проєкт курсу. Це комплексний засіб захисту інформації, що поєднує ЕЦП, шифрування та стеганографію в єдиний автоматизований конвеєр.

* `main.py` - Головний файл запуску (GUI та логіка конвеєра).
* `rsa_manual.py` - Модуль цифрового підпису (власна реалізація математики RSA).
* `xor_cipher.py` - Модуль шифрування (власний потоковий шифр XOR).
* `stego_lsb.py` - Модуль стеганографії (LSB з заголовками файлів).
* `requirements.txt` - Залежності (Pillow).

## Технічне завдання

Програма реалізує стратегію **"Defense in Depth"** (Захист у глибину), послідовно застосовуючи три методи захисту:
1.  **Цілісність (Integrity):** файл підписується приватним ключем (RSA-1024).
2.  **Конфіденційність (Confidentiality):** підписаний пакет шифрується симетричним алгоритмом.
3.  **Прихованість (Steganography):** зашифровані дані ховаються в зображення-контейнер.

**Реалізовані бонуси:**
* **Власні алгоритми:** відмова від готових крипто-бібліотек на користь авторських реалізацій.
* **Аналітика:** вбудований модуль для заміру часу виконання та зміни розміру файлів.
* **Бенчмарк:** автоматичне тестування трьох сценаріїв захисту.
* **Універсальність:** збереження та відновлення оригінального формату файлів (.docx, .jpg тощо).

**Технологія:** Python 3, Tkinter, Pillow.

## Запуск програми

1.  Встановіть залежності:
    ```bash
    pip install -r requirements.txt
    ```
2.  Запустіть додаток:
    ```bash
    python main.py
    ```
3.  **Інструкція:**
    * **Вкладка 1:** Оберіть файл і картинку, введіть пароль і натисніть "Захистити".
    * **Вкладка 2:** Оберіть захищену картинку для відновлення файлу.
    * **Вкладка 3:** Перегляньте статистику швидкодії та експортуйте звіт у CSV.
