# Лабораторна робота №2: Захист особистих повідомлень

## Опис

Ця папка містить модульний проєкт на Python, який реалізує та порівнює класичні алгоритми шифрування (Цезар та Віженер) з використанням персоналізованих ключів.

* `main.py` - Головний файл для запуску програми та демонстрації.
* `ciphers/` - Пакет з "чистою" логікою шифрів (Цезар, Віженер, спільна абетка).
* `key_generators.py` - Модуль для генерації ключів з персональних даних.
* `analysis.py` - Модуль, що реалізує бонусні завдання (Brute Force, частотний аналіз, візуалізація).
* `reporting.py` - Допоміжний модуль для форматування виводу таблиць та висновків.
* `requirements.txt` - Список зовнішніх залежностей (matplotlib).

## Технічне завдання

Програма реалізує функціонал порівняльного аналізу шифрів Цезаря та Віженера. Вона виконує наступні завдання, визначені у методичних рекомендаціях:
* Реалізація шифрування/дешифрування для **шифру Цезаря** та **шифру Віженера**.
* **Персоналізація:** Генерація ключів на основі персональних даних (прізвище та дата народження).
* **Порівняльний аналіз:** Автоматичний вивід таблиці зі стійкістю та складністю ключів.
* **Бонус (Криптоаналіз):** Реалізація функції `brute force` для автоматичного зламу шифру Цезаря.
* **Бонус (Візуалізація):** Побудова графіків частотного аналізу для трьох текстів (оригінал, Цезар, Віженер).

**Технологія:** модульний Python 3, Matplotlib (для графіків).

## Запуск програми

Програма має одну зовнішню залежність (`matplotlib`) для побудови графіків.

1.  Переконайтесь, що у вас встановлено **Python 3.x**.
2.  Перейдіть до поточної папки (`lab02`) у терміналі.
3.  Встановіть необхідні залежності (рекомендується у віртуальному середовищі):
    ```bash
    pip install -r requirements.txt
    ```
4.  Запустіть головний скрипт командою:
    ```bash
    python main.py
    ```
5.  Дотримуйтесь інструкцій програми: введіть **Прізвище**, **Дату народження** та **Текст для шифрування**.
    *(Якщо залишити текст порожнім, програма використає тестовий рядок з методички).*
